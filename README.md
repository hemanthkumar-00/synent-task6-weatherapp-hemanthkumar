# Weather Application using API Integration

# Live Demo
 https://hemanthkumar-00.github.io/synent-task6-weatherapp-hemanthkumar/
 
## Objective

To develop a weather application that fetches and displays real-time weather information using a public API.

## Steps Performed

1. Designed the application interface using HTML and CSS.
2. Integrated a public weather API using JavaScript Fetch API.
3. Displayed weather details dynamically based on user input.
4. Implemented loading states and error handling for invalid inputs.
5. Tested the application to ensure accurate data retrieval.

## Tools Used

* HTML5
* CSS3
* JavaScript
* Weather API
* VS Code
* Git & GitHub

## Outcome

A responsive weather application was successfully developed, enabling users to view real-time weather information for different locations.
